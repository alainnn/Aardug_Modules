Resource management

Easy Resource manage on task and make schedule for each

In "User Interface design" task assigns resource "Analyst", and as per time duration of that task their schedule creates in schedule tab, if the resource on leave then it will also consider in his/her scheduling...




Resource Wizard

From the resource wizard user can select resources, user can see here how many resources are available, not available and partial available, if partial available then the user can see the time how much time resource is busy...




Schedule

Resource schedule with task based on task timing, user can manage schedules automatically or manually, black color in the tree view, it means that schedule is okay, if the schedule in purple color and flag sign is there it means that resource double booked and red color means that schedule is not in the range between starting date and ending date of that task which is on schedule....




Resource Availability Report

Report shows resources availability for current week, next week, current month, next month or user can see based on custom date




Summary

Summary wizard display task, resource leaves and schedules in between time period..
